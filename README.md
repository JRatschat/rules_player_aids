# Rules and Player Aids

Just adding some rule overviews and player aids that I have created with LateX for games here. Goal is to have something ready that only contains one or two pages and can be easily printed on one page (double sided if necessary). So far, I haven't added sources to the docs, but most information stems from Shobu's [Chinese climbing game GeekList](https://boardgamegeek.com/geeklist/295046/chinese-climbing-card-games) and [pagat.com](https://www.pagat.com).
